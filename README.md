## smartbytes

> Simple cli to get airtel smartbytes balance

## Usage

Install:

```sh
npm i -D smartbytes
```

Export `SB_USERNAME` and `SB_PASSWORD`

```
export SB_USERNAME=<username>
export SB_PASSWORD=<password>
```

Run

```
$ smartbytes
```
